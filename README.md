# WMS Mobile Receipt App — Мобильная приёмка товаров

Репозиторий содержит проектные материалы, схемы и демонстрацию работы интерфейса мобильного приложения для автоматизации склада (паттерн **Offline-First**).

## 📂 Содержимое репозитория

*   Схемы процессов, архитектуры, ER-модели и Sequence-диаграммы.
*    UI-макеты экранов приложения (от авторизации до завершения процесса).
*   Видеозапись демонстрации работы пользовательского интерфейса (UI).
